# Pixel Drift

A retro-style 2-lane car dodging game built with HTML5 Canvas and vanilla JavaScript.

## How to Play

Open `index.html` in your browser — no build step required.

- **Arrow keys** or **A/D** to switch lanes
- Dodge oncoming cars to score points
- Speed increases every 20 points
- Touch support: tap left/right side of the screen on mobile

## Features

- 4 selectable car skins (yellow, blue, green, red)
- Increasing difficulty over time
- Retro pixel aesthetic with scanline overlay
